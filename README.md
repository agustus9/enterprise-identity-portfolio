# 🏢 Enterprise Identity Security Portfolio

Welcome to my Identity Governance & Administration (IGA) and Access Management portfolio. This repository documents hands-on engineering scenarios, custom code, and deployment configurations within **SailPoint IdentityIQ (IIQ)**, **Microsoft Entra ID**, and **Okta**.

## 🛠️ Global Architecture Focus
* **Governance Platform:** SailPoint IdentityIQ (On-Premises Architecture / Compliance & Lifecycle Manager)
* **Access Control:** Microsoft Entra ID & Okta Workforce Identity
* **Security Analytics:** SIEM Log Aggregation & Threat Response

---

## 📂 Portfolio Contents

### [01. Custom Rule Engineering & BeanShell Scripts](./01-IIQ-Rules-BeanShell)
* **Objective:** Developing advanced Java-based BeanShell rules to customize object behavior, data parsing, and target correlation.
* **Key Skills:** BuildMap Rules, Correlation Rules, Custom Map Rules, SailPoint API, BeanShell Scripting.

### [02. Lifecycle Manager (LCM) & Business Workflows](./02-IIQ-LCM-Workflows)
* **Objective:** Configuring and customizing XML business processes to automate complex enterprise Joiner-Mover-Leaver (JML) events.
* **Key Skills:** Workflow XML, Step Configuration, Provisioning Engine, Workitems, Lifecycle Manager (LCM).

### [03. Application Onboarding & Direct Connectors](./03-IIQ-Application-Onboarding)
* **Objective:** Integration of authoritative sources and target systems via out-of-the-box and custom application connectors.
* **Key Skills:** Schema Mapping, Active Directory Direct Connector, JDBC/Delimited File Connectors, Rule-Based Account Correlation.

### [04. Entra ID Zero Trust & Access Control](./04-Entra-Conditional-Access)
* **Objective:** Engineering Adaptive Conditional Access policies and Privileged Identity Management (PIM) to enforce least privilege.
* **Key Skills:** Entra ID, Just-in-Time (JIT) access, MFA enforcement, Zero Trust Architecture.

### [05. Okta Workforce Identity & SAML Federation](./05-Okta-SAML-Provisioning)
* **Objective:** End-to-end configuration of single sign-on federation, attribute-driven group rules, and inbound/outbound SCIM user provisioning.
* **Key Skills:** Okta Administration, SAML 2.0, SCIM Protocol, Lifecycle Management (LCM), Salesforce Integration.

### [06. SIEM Engineering & Advanced Threat Hunting](./06-SIEM-Threat-Hunting)
* **Objective:** Developing correlation logic and advanced behavioral queries within enterprise SIEM platforms to monitor identity threats.
* **Key Skills:** Splunk (SPL), Microsoft Sentinel (KQL), IBM QRadar (AQL), Log Analysis, Identity Security Monitoring.
